# Registr smluv

V Cityvizor kopíruje data z registru smluv, který je zřízen dle Zákona o registru smluv (předpis č. 340/2015 Sb.). Prokliknutím se možné se dostat do detailu jednotlivých smluv přímo v samotném registru.
